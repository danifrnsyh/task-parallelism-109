# Task Parallelism - NRP 109

Program ini merupakan implementasi task parallelism menggunakan Python.

## Penjelasan
Program menjalankan 3 tugas berbeda secara paralel:
- Fetch data
- Compute data
- Save data

Menggunakan threading sehingga ketiga proses berjalan bersamaan.

## Cara Menjalankan
python main.py
